EasyXL - 让AI帮你完成数学作业

发表于 2026-04-02 分类于 JS脚本 阅读次数：

在学校经常被数学老师布置的IXL作业所困扰，于是我推出了IXL的辅助答题工具——EasyXL，通过读取网页内容并发送给AI以获取IXL题目的答案。

为什么开发这个工具？

完成IXL作业十分繁琐费事，也有不少学生被IXL的题目困扰过——答对得2分、答错扣10分的机制是许多人憎恨IXL的主要原因，而我也这么认为。

许多学生急需一个辅助解题的方式，以此解放我们在IXL上浪费的时间，而在当今AI的加持下，这种渴望成为了现实！

如何使用

Github源址：https://github.com/tev6/EasyXL

使用该工具需要用到油猴 Tampermonkey 插件，请将其下载到你的游览器，然后使用以下步骤：

  1. 从仓库中复制easyxl.js的源代码
  2. 打开Tampermonkey的面板，点击：添加新脚本
  3. 全选编辑器中已有的代码，并Ctrl+V将刚才复制的js代码覆盖到脚本中
  4. Ctrl+S保存更改
  5. 退出Tampermonkey的页面，你现在可以打开IXL网页进行测试了

API设置

该工具需要使用到API，这里支持Deepseek, Openai, Anthropic，Kouri AI的API供应商，你也可以自定义Base URL。

在EasyXL窗口右上角的设置图标中进行操作，选择你的供应商，并提供你的API Key（这会在本地保存）和模型选择。

如果这一切都设置正确，那么你就可以在IXL答题页面上测试了，点击”Parse and Solve”按钮就能将题目发送请求给AI进行解决，他会返回题目的答案。

IXL示例图片

功能特点

  • 可拖拽的 IXL 页面悬浮窗，含有半透明模糊玻璃样式效果
  • 单独按下并松开 Ctrl 可隐藏/显示窗口（不会影响 Ctrl+C 等组合键）
  • 单一主脚本内集成 OpenAI、Anthropic、Google、DeepSeek、Kouri 多供应商切换
  • 用户注释功能：允许在解析前添加自定义提示或解题要求的备注，这些备注将会发送给 AI。
  • 使用 GM_xmlhttpRequest 发送跨域请求以绕过页面 CSP 限制
  • 支持在结果框中渲染富文本 Markdown 和数学公式（基于 KaTeX）
  • 右上角齿轮按钮可打开设置面板，支持退出按钮、ESC、点击遮罩关闭
  • 为每个供应商分别保存 API Key、模型、基础 URL 和用户注释
  • 请求失败或配置无效时，会在设置面板中给出提示并引导重新配置

技术原理

程序直接将题目范围内的HTML内容发送，并使AI提取其中的题目内容后再进行作答，程序会要求AI给出解题思路（此步可以提高正确率），但是强制要求将最终的精简答案输出于<answer>...</answer>当中，这样程序之后提取当中的内容并反馈给客户。
